# ADR-002: bleed 자동 연장은 래스터 방식으로 (프로토타입)

- 상태: 채택 (2026-07-03)

## 결정

`extend_bleed`는 페이지를 300dpi로 래스터화한 뒤 가장자리 픽셀을 연장(edge replicate)하고
CMYK 이미지로 PDF를 재조립한다. 원본 파일은 보존한다 (reversible).

## 근거

벡터 보존 방식(콘텐츠 스트림의 오브젝트를 실제로 늘리는 것)은 디자인 의도 해석이 필요해
프로토타입 범위를 넘는다. 래스터 방식은 90x90mm 스티커 기준 수 초 내 동작하고
전/후 비교 이미지를 바로 만들 수 있어 시연 요건을 충족한다.

## 한계와 본개발 과제

1. 벡터/텍스트가 이미지가 된다 → 본개발에서 벡터 보존 방식으로 교체.
2. 색 변환이 PIL 나이브 CMYK(잉크 총량 최대 300%)다 → ICC(GRACoL/JapanColor) 기반으로 교체.
3. A2 포스터급은 래스터 용량이 커져 느리다 → 본개발에서 해상도 정책 분리.
