# 애즈랜드 작업 가이드 — 검수 반영 현황 & 정직한 한계

애즈랜드(adsland.com) 작업 가이드를 검수 규칙으로 반영한 결과다. **자동으로 확인되는 것**과
**파일만으로는 확인이 불가능하거나 아직 못 한 것**을 솔직하게 구분해 적는다. (요청: "못 하는 부분은
정직하게 적어놔줘.")

가이드 출처(2026-07 확인, 코드의 `core/preflight/adsland_guide.py` `GUIDE_URLS`):
일러스트 `guide_01_01_ill_win` · 포토샵 `guide_01_01_pho_win` · 인디자인 `guide_01_01_ind` ·
PDF `guide_01_03_prf` · 품목별 `guide_01_02_*` · 후가공/도무송 `guide_03_*` · 용지 `guide_04_01`

---

## 1. 자동으로 검수·반영되는 항목 (구현 완료)

| 검수 항목 | 가이드 기준 | 반영 |
|---|---|---|
| 색상 모드(colorspace) | 문서·오브젝트 CMYK 필수 (RGB·미변환 별색 금지) | RGB/별색 검출 → 경고 + RGB는 자동 CMYK 근사 변환 |
| 해상도(resolution) | 인쇄 이미지 300dpi 이상 | 이미지 유효 dpi 측정, 150~300 warn / 150 미만 fail |
| 재단여백(bleed) | **품목별**: 명함 1mm / 전단 2mm / 스티커 3mm | 품목 기준으로 판정 + 규격보다 큰 여분은 재단여백으로 인정 + 부족분 자동 보정 |
| 안전여백(trim_safety) | **품목별**: 명함·사각스티커 2mm / 도무송·리플렛 3mm | 텍스트가 안전영역 침범 시 고지 |
| 총잉크량(ink_total) | **품목별**: 전단·포스터·리플렛 250% / 그 외 300% | 벡터·텍스트 CMYK 합 측정 (이미지 픽셀 제외) |
| 검정 텍스트(black_type) | 일반 텍스트 K100(먹1도), 4도 블랙 지양 | 텍스트에 CMY 섞였는지 검출 |
| 최소 선굵기(min_line) | 0.5pt 미만 선 지양(점선처럼 끊김) | 0.5pt 미만 스트로크 warn (칼선 별색 제외) |
| 글꼴(font_embed) | 모든 문자 아웃라인/임베드 | 미임베드 폰트 검출 (아웃라인 권장 안내) |
| 투명도·오버프린트(transparency) | 투명 병합, 오버프린트 해제 | 투명·오버프린트 플래그 검출 |
| 재단 크기(page_size) | 파일 재단 크기 = 주문 규격 | 규격 대조(재단여백 감안) |
| 페이지 수(page_count) | 단면 1 / 양면 2, 앞·뒤 별도 | 주문 면 수와 대조 |
| 칼선 존재(dieline) | 도무송은 칼선 필요 | 별색 칼선 존재 여부(아래 한계 참조) |

문제 항목마다 **가이드 근거 + 왜 + 프로그램별 고치는 법 + 가이드 링크**를 검판 카드에 함께 보여준다
(`fix_guide`). 자동 보정 가능(도련·색상)은 버튼으로, 불가능은 "원본에서 이렇게 고쳐 다시 올려주세요".

## 2. 파일(PDF)만으로는 확인이 **불가능**한 것 — 구조적 한계

1. **도무송 칼선 검출 방식 차이 (중요).**
   애즈랜드는 `CutContour` 같은 **명명된 별색을 쓰지 않는다.** 도무송 칼선을 **검정 K100%**로
   그리고, 파일을 `도무송 위치용 / 인쇄용 앞(칼선 색없음) / 인쇄용 뒤 / 도무송 도안(K100%)`
   **4개로 분리**해 접수받는다(출처 `guide_03_03`). 하나로 합쳐진 PDF 한 장에서는
   - K100 검정은 본문에도 쓰이므로 "이 K100이 칼선인지" 자동 구별이 사실상 불가.
   - 4-파일 분리 구조를 단일 파일 검수로는 확인 불가.
   → 현재는 **고객이 고른 재단 형태(cut_type)**를 신뢰하고, 별색 칼선이 있으면 그것으로 판단한다.
     진짜 도무송은 칼선 파일을 별도로 받아 사람이 확인하는 흐름이 정확하다.

2. **K100 자동 오버프린트.** K100 개체는 **출력(RIP) 시점에 자동 오버프린트**된다 — 파일 안에
   플래그로 남지 않아 파일 검사로는 잡히지 않는다. 가이드 권고(K100+C1 또는 K99)는 안내만 한다.

3. **레이어 상태(잠금·숨김·미사용 레이어).** 인쇄용 PDF는 대개 병합(플래튼)돼 원본 레이어
   상태가 남지 않는다 → 자동 확인 불가. 안내로만.

4. **글꼴 "아웃라인" vs "임베드".** 가이드는 아웃라인(윤곽선화)을 요구하지만, 임베드된 폰트는
   우리 검사를 통과한다. 임베드는 잡되 "아웃라인 안 됨"은 강제하지 않는다(자동 아웃라인은
   오타·글리프 깨짐 위험이 커 자동 적용하지 않음 — 안내만).

5. **이미지 픽셀 단위 잉크량.** 총잉크량은 벡터·텍스트 CMYK만 측정한다. 사진(래스터) 내부
   픽셀별 잉크량은 프로토타입 범위 밖.

## 3. 아직 **구현 안 한** 것 (가능하지만 미착수) — 정직하게

- **CMYK 프로파일 지정 확인**: 포토샵 가이드는 `CMYK Japan Color 2001 Coated`를 명시하지만,
  우리는 RGB/CMYK 여부만 보고 특정 ICC 프로파일까지는 확인하지 않는다.
- **PDF 버전/규격 확인**: 포토샵=Acrobat5(PDF1.4), 인디자인=PDF/X-1a:2001 권고를 파일 버전으로
  검사하지 않는다.
- **후가공 위치 도안(박·형압·타공 등) 검수**: 후가공은 "인쇄 도안 + 가공 위치 도안(K100)"
  분리 방식(형압 선 1pt·텍스트 8pt 등)인데, 후가공 파일 검수는 아직 없다.
- **품목별 표준 규격 크기(mm) 대조**: 아래 4번 참조.

## 4. 표준 규격 크기(mm) — 다운로드로 **대부분 확보 완료**

가이드 안내 페이지는 이미지였지만, **주문 페이지의 규격 spec 테이블(HTML)**을 직접 받아 파싱해
실제 재단 크기를 확보했다(코드 `adsland_guide.py`의 `STANDARD_SIZES`). 확보된 값:

- **명함/엽서**: 자유 입력(재단 **85×45 ~ 150×100mm**), 표준 프리셋 **90×50 · 86×52 · 88×54**,
  도련 1mm. → **카탈로그 명함 사이즈를 86×54 → 86×52로 가이드에 맞춰 정정함.**
- **떡메모지**: 재단 80×80·90×90·100×100·100×140·A4 등, 도련 2mm·안전 2mm.
- **전단/포스터**: A1(600×900)~A6(103×146), B2~B7 전 규격, 도련 2mm·안전 2mm.
- **사각재단 스티커**: 50×50·80×50·90×55~150, 편집=재단+3mm.
- **도무송 스티커**: 칼선=재단−3mm, 편집=재단+3mm, 최소 10mm, 도형 간 3mm.
- **메탈** 18×20~197×217mm, **에폭시** 15~100mm.

남은 한계: **다운로드 가능한 칼선 템플릿 파일(.ai/.psd/.zip)은 공개 링크로 존재하지 않았다**
(사이트가 3선 도안 이미지 `kalimg/standard/*.png` + 톰슨 가이드로만 제공). 칼선 규칙(K100·최소10mm·
간격3mm·4파일 분리)은 이미지 OCR로 확보해 `DIELINE_RULES`에 기록. 카탈로그/가격표를 애즈랜드
**전 규격**으로 넓히는 것(각 규격의 가격 매핑 포함)은 후속 작업으로 남긴다.

## 5. 자동 보정(autofix) 현황

| 항목 | 자동 보정 | 방식 |
|---|---|---|
| 재단여백 부족 | ✅ | 가장자리 픽셀 복제로 품목 기준 도련 생성(색 유지) |
| RGB → CMYK | ✅ | 300dpi 래스터 후 GCR 변환(검정에 K). ICC 없이 근사라 '색 약간 달라질 수 있음' 고지 |
| 총잉크량 초과 | ❌ | 안내만 (자동 잉크 리미팅은 색 왜곡 위험) |
| 글꼴 아웃라인 | ❌ | 안내만 (자동 아웃라인은 글리프 깨짐 위험). 아웃라인 안 됨은 '참고' 안내로 표시 |
| 오버프린트/투명 | ❌ | 탐지·안내만 (자동 병합/해제는 결과 예측 어려움) |

색·잉크·글꼴을 함부로 자동 변경하지 않은 이유: **잘못 건드리면 실물이 상한다.** 위험한 항목은
"이렇게 고쳐 다시 올려주세요"로 가이드 근거와 함께 안내한다.

## 6. 추가 구현 (파일 형식·이미지·칼선)

- **JPG/PNG 접수**: 이미지를 주문 규격 크기의 PDF로 감싸 검수(해상도·크기 유효). 벡터 검수
  (선·별색·글꼴)는 이미지엔 해당 없음을 '참고'로 안내. (그 외 AI/PSD/EPS/CDR은 아직 미지원 —
  PDF로 저장해 올려야 함.)
- **오버프린트 탐지**: ExtGState의 `/OP`·`/op` 설정을 잡아 경고. (K100 RIP 자동 오버프린트는
  파일에 없어 여전히 안내만.)
- **도무송 칼선 별도 업로드**: 도무송인데 칼선이 없으면 'K100 칼선 파일'을 따로 받아 검증
  (벡터 선 존재 + 최소 10mm) 후 통과. **한계**: 애즈랜드 실제 칼선 샘플 미확보로 3R·간격 등
  정밀 규칙은 미검증(휴리스틱). 4파일 분리 접수(앞/뒤/위치/칼선)의 완전 검수도 샘플 확보 후.
